## 描述

當要以Hash來儲存以下資料時，且該使用者的ID是1

```
{
  "name": "apple",
  "age": 25
}
```

會是以


```
{
  "name": "apple",
  "age": 25,
  "clothes": {
    "shirt": "gray",
    "pants": "read"
  }
}
```
