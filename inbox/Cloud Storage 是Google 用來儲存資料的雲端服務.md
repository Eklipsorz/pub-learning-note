**Cloud Storage**

-   **Cloud Storage**：  
    適用於各種不同規模企業的物件儲存空間，且無資料擷取次數或頻率限制
-   **Cloud Datastore**：  
    可擴充的 NoSQL 資料庫，適用於網頁和行動應用程式
-   **Cloud Bigtable**：  
    可擴充、全代管的 NoSQL 寬欄資料庫服務，適合處理大型分析和大量低延遲工作負載，可用性高達 99.999%
-   **Cloud SQL**：  
    MySQL、PostgreSQL 和 SQL Server 皆適用的全代管關聯資料庫服務
