

## 描述
引用於[[@geeksforgeeksWhatNetworkID2021]]所描述
IPv4上的序列分為兩個部分：
- Network ID
- Host ID

### Network ID 
Network ID表明目前序列IP所屬的子網域或者網域是什麼

### Host ID
Host ID 表明目前序列IP在所屬的子網域/網域上的獨特序列是什麼，也就是在這網域用來識別目前主機的序號是什麼
