## 描述
根據[[@fasterrizeCacheWarmingWhy]]所描述：
> Warming up the cache involves **artificially filling the cache so that real visitors always have cache access**

重點：
- Warmup the cache 是指事先插入常出現的資料至緩存來增加cache hit機率，從而增加效率
